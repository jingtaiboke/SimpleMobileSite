# SimpleMobileSite
#### 描述<br>
这是一个代码量极小的，适用于PC、手机浏览器，以及微信等内嵌浏览器APP的响应式网站模板。

#### 适合场景<br> 
手机游戏平台，娱乐、新闻平台等

#### 开发应用<br>
本模板经过良好的兼容性测试，可以用于自己开发的威望张，也可以直接应用于hybird app开发中用web view加载
